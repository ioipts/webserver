# Web server

Muti-threading web/api server  
Support upload/download file

## Feature
Written in C/C++11.  
Support multi-platform.  
Support both 32-bit and 64-bit.  

## How to use
Just copy axishttpsock.cpp, axishttpsock.h and include in the project.  
Enjoy!  

## How to run the sample
cd sample  
make  
./webserver
